# Jokiin - Website Jasa Pengerjaan Tugas

Jokiin adalah website yang menyediakan layanan jasa pengerjaan tugas akademik profesional. Website ini dibangun menggunakan HTML, CSS, dan JavaScript dengan desain modern dan responsif.

## Fitur

- Desain modern dan responsif
- Animasi smooth menggunakan AOS (Animate On Scroll)
- Slider testimonial menggunakan Swiper.js
- Form validasi
- FAQ accordion
- Rating system
- Smooth scroll
- Mobile-friendly navigation
- Scroll to top button

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript (ES6+)
- [AOS Library](https://michalsnik.github.io/aos/)
- [Swiper.js](https://swiperjs.com/)
- [Font Awesome](https://fontawesome.com/)
- [Google Fonts](https://fonts.google.com/)

1. Clone repository ini
2. Buka file `index.html` di browser Anda
3. Atau gunakan live server di editor Anda
